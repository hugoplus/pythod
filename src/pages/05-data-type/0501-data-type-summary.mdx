## 基本概念

除了我们在第四章中学到的一些基本类型外，Python 还提供了多种数据类型，以便于程序开发。

这些数据类型可以分为以下几类：

-   数字类型：包括整数、浮点数和复数。
-   字符串类型：表示文本信息的数据类型。
-   字节类型：表示字节序列，序列中的每一元素是一个字节（Byte）对象。
-   布尔类型：表示真假的数据类型，只有两个值：`True` 和 `False`。
-   列表类型：表示一组有序的数据的数据类型，可以存储不同类型的数据。
-   元组类型：表示一组有序的数据的数据类型，可以存储不同类型的数据，但是一旦创建就不能更改。
-   字典类型：表示一组无序的键值对的数据类型，可以存储不同类型的数据，每个元素都由一个键和一个值组成。
-   集合类型：表示一组无序的数据的数据类型，可以存储不同类型的数据。
-   冻结集合类型：表示一组无序的数据的数据类型，可以存储不同类型的数据，但是一旦创建就不能更改。
-   函数类型：表示一个可以执行特定任务的代码块的数据类型。
-   类类型：表示一组数据和方法的数据类型，可以用于创建对象。

以上就是 Python 中常见的数据类型，它们在编写 Python 程序时都会经常使用到。

我们会在后续章节中对它们进行详细解释。
